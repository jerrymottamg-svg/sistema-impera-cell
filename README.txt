🚀 COMO USAR
─────────────
1. Abra o arquivo "impera_cell.html" com o Google Chrome ou Edge
2. Os dados são salvos automaticamente no navegador (localStorage)
3. IMPORTANTE: sempre use o mesmo navegador no mesmo computador
   para que os dados persistam entre sessões.


💾 BACKUP (MUITO IMPORTANTE)
──────────────────────────────
O sistema salva os dados no navegador, mas esses dados podem ser
perdidos se o navegador for limpo ou o computador formatado.

▶ COMO FAZER BACKUP:
  1. Na barra verde do sistema, clique em "⬇ Exportar backup"
  2. Um arquivo .json será baixado automaticamente
  3. Mova esse arquivo para a pasta:
     C:\Users\teste\Documents\Nova pasta\Impera Cell\backups\
  4. Recomendado: fazer backup todo final de dia

▶ COMO RESTAURAR BACKUP:
  1. Na barra verde, clique em "📂 Importar backup"
  2. Selecione o arquivo .json mais recente da pasta backups\
  3. Confirme — todos os dados serão restaurados


🔄 RECEBER ATUALIZAÇÕES DO SISTEMA
────────────────────────────────────
Quando uma nova versão do impera_cell.html for enviada:
  1. Faça um backup ANTES de substituir (Exportar backup)
  2. Substitua o arquivo impera_cell.html pela nova versão
  3. Abra o novo arquivo e importe o backup salvo
  → Todos os dados voltam normalmente
